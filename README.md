# Codebar

A creative technology studio landing page built with HTML, CSS, JavaScript, and Vite.

## Features

*   **Responsive Design:** Fully responsive across mobile, tablet, and desktop viewports.
*   **Modern Aesthetics:** Apple Siri-inspired design system with glassmorphism, gradients, and dynamic glow effects.
*   **Interactive Particle Wave:** A custom WebGL Three.js particle wave effect in the hero section that flows continuously and reacts to mouse movement.
*   **Scroll Animations:** Smooth scroll reveals and sticky showcase scroll-linked animations.
*   **Interactive Elements:** Hover-tilt 3D effects on service cards, dynamic counters, and an animated AI orb.
*   **Floating Contact:** Quick access WhatsApp and Phone buttons.

## Tech Stack

*   **Core:** HTML5, Vanilla JavaScript (ES Modules)
*   **Styling:** Vanilla CSS3 with CSS Variables
*   **Graphics:** Three.js (WebGL) for the hero particle effect
*   **Build Tool:** Vite

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1.  Clone the repository or download the source code.
2.  Navigate to the project directory:
    ```bash
    cd codebar
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Development Server

Start the local development server with Vite:

```bash
npm run dev
```

Navigate to `http://localhost:5173/` in your browser.

### Building for Production

To create a production-ready optimized build:

```bash
npm run build
```

This will output the static files into the `dist/` directory.

You can preview the production build locally:

```bash
npm run preview
```
